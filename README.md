# Fireline - Интернет-магазин биокаминов премиум-класса

Современный, быстрый и SEO-оптимизированный интернет-магазин, разработанный с использованием передовых технологий веб-разработки.

## 🚀 Технологический стек

### Frontend
*   **Next.js 15 (App Router)**: Основа проекта. Используются серверные компоненты (Server Components) для мгновенной загрузки и клиентские компоненты для интерактива.
*   **TypeScript**: Обеспечивает типизацию кода, минимизируя ошибки и улучшая поддержку проекта.
*   **Tailwind CSS**: Используется для быстрой и гибкой верстки адаптивного дизайна.
*   **NextUI**: Библиотека современных UI-компонентов (кнопки, модальные окна, инпуты).
*   **Framer Motion**: Плавные анимации элементов интерфейса и переходов.
*   **Lucide React**: Набор легких и красивых иконок.

### State Management
*   **Redux Toolkit**: Управление глобальным состоянием (корзина, список товаров).
*   **Redux Saga**: Обработка побочных эффектов (асинхронные запросы к API, синхронизация с localStorage).

### Backend & CMS
*   **Sanity CMS**: Мощная headless-система управления контентом. Позволяет удобно добавлять товары, категории, характеристики и изображения.
*   **Upstash Redis**: Высокопроизводительное кэширование API-запросов для снижения нагрузки на CMS и ускорения работы сайта.
*   **Next.js API Routes**: Серверная логика для обработки запросов и интеграции с внешними сервисами.

## 🛠 Реализованный функционал

1.  **Каталог и Категории**:
    *   Динамический вывод категорий из Sanity.
    *   Фильтрация товаров по категориям.
    *   Живой поиск товаров по названию в хедере.
2.  **Карточка товара**:
    *   Серверный рендеринг (SSR) для мгновенного отображения.
    *   Интерактивный слайдер изображений.
    *   Вывод характеристик и детального описания.
    *   Динамические SEO-метаданные для каждой страницы.
3.  **Корзина**:
    *   Добавление/удаление товаров, изменение количества.
    *   Сохранение состояния корзины в браузере (Persistence).
    *   Синхронизация состояния кнопок "В корзину" на всех страницах.
4.  **Система заказов**:
    *   Форма "Заказ в один клик" (Callback Modal).
    *   Интеграция формы на странице товара и в корзине.
5.  **SEO & Оптимизация**:
    *   Автоматическая генерация `sitemap.xml`.
    *   Оптимизация изображений через `next/image`.
    *   Кэширование данных на стороне сервера.

## 📱 Адаптивность
Сайт полностью адаптирован под мобильные устройства и планшеты, включая удобное бургер-меню и перестраиваемую сетку товаров.

## 🏁 Запуск проекта

1.  Установите зависимости: `npm install`
2.  Настройте переменные окружения в `.env.local` (Sanity Project ID, Dataset, Redis URL).
3.  Запустите сервер разработки: `npm run dev`
4.  Откройте [http://localhost:3000](http://localhost:3000)

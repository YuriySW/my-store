# Ключевые изменения

- Добавлена настройка `preview` в схему `category.ts` для корректного отображения названий категорий в Sanity Studio.
- Добавлена настройка `preview` в схему `product.ts` для отображения названия, цены, категории и изображения товара в списке админки.
- Исправлено дублирование Header и Footer на всех страницах (удалены локальные импорты в пользу глобальных в layout.tsx).
- Очищены файлы страниц: contacts, news, payment, delivery, portfolio, cart.
- Обновлены стили пустой корзины (текст и иконка теперь серые).
- Заголовки информационных страниц (Портфолио, Новости и др.) изменены на серый цвет.
- Созданы новые страницы: "Дизайнерам" (`/designers`) и "О компании" (`/about`).
- Обновлена страница контактов с актуальными данными (адрес, почта, время работы).
- Исправлены ссылки в футере.
- Улучшено мобильное меню: вся область "Каталог" теперь кликабельна, исправлены цвета стрелочки и разделителей, добавлено корректное сворачивание при повторном клике.
- Исправлен курсор для десктопного меню "Каталог".
- Мобильный поиск теперь открывается под логотипом, не перекрывая его.
- Исправлена навигация в мобильном меню: клик по категории закрывает всё меню.
- Разделены состояния мобильного и десктопного каталога для стабильной работы.
- Исправлено "мигание" и блокировка кликов в мобильном меню.
- Добавлена загрузка корзины из localStorage при инициализации страницы (товары не исчезают после обновления).
- Исправлены мобильные стили корзины: улучшены отступы, размеры шрифтов и изображений для мобильных устройств.
- Добавлены padding top и bottom 7px для кнопок "В корзину" и "Заказать звонок" на странице товара.
